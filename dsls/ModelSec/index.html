ModelSec

ModelSec offers a generative architecture for managing security requirements. This architecture automatically generates security software artifacts (e.g., security rules) through a model transformation chain comprising two steps. First, a security infrastructure-dependent model is derived from three models, which express the security restrictions, the design decisions, and the information needed on the target platform. Subsequently, the generated model is transformed into security software artifacts. http://dx.doi.org/10.3217/jucs-015-15-2957
