Pavlich-Mariscal et al.

To assist designers to visualize access control policies separated from non-security concerns, this paper proposes a set of access control diagrams, i.e., extensions to the UML to represent three main access control models: role-based access control (RBAC), mandatory access control (MAC), and discretionary access control (DAC). - https://doi.org/10.1016/j.cose.2009.11.005
