CCMSec

An MDA tool chain that supports the rapid model-based development process for security-critical software systems, starting with the definition of secure system models in high-level modeling languages such as UML, and then automatically transforming them into secure target system models such as CCM, which can be used for further steps such as code generation of CCM components with security properties or simple security policy generation. - https://doi.org/10.1016/j.entcs.2006.10.016
