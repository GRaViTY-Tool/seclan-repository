Zhu et al.

A model-based aspect-oriented framework for building intrusion-aware software systems. Attack scenarios and intrusion detection aspects are modeled using an aspect-oriented Unified Modeling Language (UML) profile. Based on the UML model, the intrusion detection aspects are implemented and woven into the target system. The resulting target system has the ability to detect the intrusions automatically. - https://doi.org/10.1016/j.infsof.2008.05.007
