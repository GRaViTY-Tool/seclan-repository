SSD RBAC

An aspect-oriented modeling (AOM) approach for addressing access control concerns. In the AOM approach, an aspect addresses a pervasive access control concern. Other functional design concerns are addressed in a model of the application. Composing access control aspects with a primary model results in an application model that addresses access control concerns. http://dx.doi.org/10.1016/j.infsof.2003.10.007
