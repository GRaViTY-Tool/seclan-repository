UMLsec

UML extension for secure systems design. It uses standard UML extension mechanisms and can be used to evaluate UML specifications for vulnerabilities using a formal semantics of a simplified fragment of the UML. Established security engineering rules can be encapsulated and made available to developers who are not security specialists.
